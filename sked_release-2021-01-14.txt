Release notes for sked
2021-01-14
John Gipson

For more details about the changes read chnage_log.txt in the sked 
source directory.

If you have installed sked previously you should have no problems
installing this version. The steps are:

1. From previous installations you should have a file like 
"set_misc_gfortran64". This contains information about the location 
of libraries etc. If this is the first time you have installed
sked you may need to modify this.

2. Modify the file sked/skedf_ctl.ftni to point to your copy of skedf.ctl.

3. In the sked source directory type "source set_misc_gofortran64"

4. Type make_sked. This should make all of the libraries and sked. 

----Summary of changes----
Most changes should be transparent to the user EXCEPT the following:

The location of the global skedf.ctl file is contained in 
'sked/skedf_ctl.ftni'.
You will need to modify this to point to the correct location on 
your PC.

Other changes:

1.) A lot of obsolete code dealing with tapes was removed. There is 
still a lot more to be done. One consequence is that some sections of
the SKD and VEX files are no longer written, e.g., information about 
the head position. 

2.) Unification of the skdrut libray  between FS and sked. 
The library skdrut.a contains utilties that are used by both sked and drudg. 
In the past year some development work was done on drudg independently. 
Because of this the versions used by sked and drudg diverged. This version
unifies them.
